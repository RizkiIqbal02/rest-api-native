Dibuat dengan penuh kegabutan maksimal, maaf pak kalau source code saya banyak hal-hal yang tidak semestinya di masukkan hehe, pertama kali saya coba laravel langsung suka, jadinya kalau ngoding struktur foldernya suka di mirip miripin laravel aja hehe.


Fitur-fitur yang sudah aku include yaitu:

- Migration database kayak laravel, dengan klik tombol Migrate schema yang akan digunakan pada REST API ini akan otomatis di buatin.
- Konfigurasi MySQL di dalam file .env, Sesuaikan host dan port MySQL agar lebih rapih dan mudah, aku pisahin di dalam .env.
- Klik Migrate otomatis di buatkan tabel nya, tapi ngga beserta data awalnya, bisa ditambahkan langsung sendiri aja lah ya, sekalian test REST API nya hehe.
- Struktur folder nya di sesuaikan dengan fungsinya, agar lebih mudah dalam kerja tim atau hanya sekedar debugging.
- Layout ramah user dan mudah di mengerti (Harapan ku).
- Sudah disertakan Docs nya lengkap dengan METHOD-METHOD yang bisa di gunakan.

Cara penggunaan REST API ini :
- Jalankan WEB SERVER dan juga Start MYSQL nya.
- Tentukan nama database MYSQL yang ingin kamu buat ke database (DB_DATABASE)
- Jalankan WEB SERVER dan arahkan ke folder project ini ( htttp://localhost/rest-api-native )
- Klik tombol Migrate dan Schema tabel nya akan otomatis di buat.
- REST API siap di gunakan, untuk petunjuk METHOD-METHOD yang bisa digunakan, klik tombol API DOCS
